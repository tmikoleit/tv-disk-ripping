# Disk Ripping Automation

Python-based tool for automatic episode matching and file organization for MakeMKV rips.

**See [Operational/README.md](Operational/README.md) for complete documentation.**

## Quick Start

```bash
cd Operational
pip install -r requirements.txt
python process_rips.py [show] [season] [disk]
```
